## Key Decisions

| Decision | Rationale | Trade-offs |
|----------|-----------|------------|
| Event-driven import (SQS) | WAF bypass, async processing, DLQ visibility | Added complexity (polling, status endpoint) |
| 10KB file size threshold | Forces presigned URL for most CSVs | S3 storage cost (~$0.023/GB/mo) |
| Idempotency key pattern | Safe retries, no duplicate rows | DynamoDB storage + TTL management |
| OpenAPI-first | Frontend parallelizes, contract testing | Spec maintenance overhead |

## Cost Impact

| Change | Cost Impact |
|--------|-------------|

**Total incremental:** ~$60–70/mo for production hardening.
